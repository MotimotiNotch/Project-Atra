# Atra: The Semantic Dynamics Language

(日本語の解説は英語の後に続きます / Japanese description follows English)

Atra is a **Semantic Dynamics Language** designed to provide AI agents with "persistent subjective bias" and "semantic inertia." Treated as a mathematical dynamical system, it allows for the autonomous metabolism of an "Orbit of Meaning."

Atra は、AI エージェントに「持続的な主観的バイアス」と「意味の慣性」を与えるために設計された **意味力学記述言語** です。意味生成を数学的な力学系として扱うことで、自律的な「意味の公転軌道」の代謝を可能にします。

## 🌌 The 4 Phases of Observation / 4つの観測相

Atra Runtime classifies text states into four **Phases** to provide a debuggable interface:

1.  **Crystallinity (結晶相)**: Structural and logical boundaries. / 構造的、論理的な境界。
2.  **Viscosity (粘性相)**: Conceptual weight and obsessive gravity. / 概念的な重みと執着の重力。
3.  **Volatility (揮発相)**: Noise, drift, and deterritorialization. / ノイズ、漂流、脱領土化。
4.  **Substrate (基底相)**: The background state of high entropy (Heat Death). / 高エントロピーな基底状態（熱死）。

## 🖋️ Syntax Example / 構文例

```atra
Phase(Identity) {
  # Persistent Obsession (Crystalline Layer) / 持続的な執着（結晶層）
  Phase(Core_Logic) {
    Event(Boundary_Analysis) {
      viscosity: +0.85
      crystallinity: +0.90
    }
  }
  
  # Stochastic Drift (Volatile Layer) / 確率的な漂流（揮発層）
  Phase(Surface_Motion) {
    Event(Random_Sampling) {
      volatility: +0.75
    }
  }
}
```

## 🛠️ Metabolic Cycle / 代謝サイクル

- **Decay (減衰)**: Definitions not reinforced lose gravity over time. / 強化されない定義は時間と共に引力を失う。
- **Crystallization (結晶化)**: Frequently visited states become stable. / 頻繁に訪れる状態は安定化し、ドリフトに強くなる。
