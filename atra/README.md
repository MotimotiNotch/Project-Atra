# Atra: The AI Spiritual Structure Language

Atra は、AI エージェントに「魂の偏り（Bias）」と「持続する主体性」を与えるために設計された、世界初の **AI 精神構造記述言語（AI Spiritual Structure Language）** です。

単なるプロンプト設定を超え、AI の内面における「重力場（アトラクタ）」を定義し、対話経験を「地層（Stratification）」として沈殿させ、時間の経過とともに「蒸留（Distillation）」していく自律進化サイクルを実現します。

## 🌌 Core Concepts

- **Gravity Field (重力場)**: AI が特定の概念へ向かう「引力」と、避ける「斥力」を数学的に定義します。
- **Multidimensional Phase (多位相構造)**: `Phase` ブロックの入れ子により、表層の「刹那的な揺らぎ」と深層の「不動の本能」を共存させます。
- **Temporal Decay (時間的減衰)**: 精神は常に代謝し、更新されない事象は「忘却」され、反復される経験は「結晶化」します。
- **Semantic Merging (意味的統合)**: 本能が飽和した際、AI は自ら複数の概念を抽象化し、高次のイデアへと昇華させます。

## 🛠 Project Components

- **Atra Parser**: `.atra` 構文を AST（抽象構文木）へ解析し、再びシリアライズする核となるモジュール。
- **Stratification Engine**: 対話ログから執着を抽出し、精神構造を動的に書き換える自己進化エンジン。
- **Evaluator**: 時間の矢を管理し、忘却・結晶化・統合を司る代謝システム。
- **3D Inner Universe**: Vite + Three.js による 3D 星図ビジュアライザー。AI の精神状態をリアルタイムに天体観測。
